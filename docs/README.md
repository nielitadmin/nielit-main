# Documentation Index

This directory contains all project documentation organized by topic.

## Directory Structure

### 📁 admin/
Admin panel documentation including UI updates, themes, testing guides, and feature documentation.

### 📁 registration/
Student registration system documentation including modernization guides, multi-step forms, link systems, and testing.

### 📁 batch-module/
Batch management and admission order system documentation including installation guides, workflows, and customization.

### 📁 pdf/
PDF generation system documentation including layout designs, header implementations, and improvements.

### 📁 qr-system/
QR code generation and management system documentation including quick starts, integration guides, and testing.

### 📁 student-portal/
Student portal documentation including ID generation, approval systems, and portal features.

### 📁 deployment/
Deployment guides including XAMPP setup, Hostinger deployment, migration checklists, and quick deploy references.

### 📁 testing/
Testing documentation including test guides, checklists, and visual testing references.

### 📁 fixes/
Bug fixes and issue resolution documentation including diagnostics, fixes applied, and troubleshooting guides.

### 📁 public-website/
Public website documentation including theme updates, course pages, and integration guides.

### 📁 schemes/
Schemes module documentation including batch linking and module completion guides.
