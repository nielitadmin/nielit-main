## 🔧 Troubleshooting

### Images Not Showing?
**Check**: `assets/images/` folder has:
- `bhubaneswar_logo.png`
- `National-Emblem.png`

### CSS Not Loading?
**Check**: `assets/css/public-theme.css` exists
**Verify**: `APP_URL` in `config/config.php`

### Navigation Not Working?
**Check**: Bootstrap 5 JS is loaded at bottom of page

### Map Not Loading?
**Check**: Internet connection (Google Maps needs internet)
