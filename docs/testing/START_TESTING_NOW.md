## 🐛 Common Issues (Quick Fixes)

### "Progress not updating"
**Fix:** Refresh page (F5), check browser console

### "Validation not working"
**Fix:** Use valid test data first, check field names

### "File preview not showing"
**Fix:** Check file size < 5MB, verify file type

### "Animations choppy"
**Fix:** Close other tabs, use Chrome/Firefox

### "Mobile layout broken"
**Fix:** Press F12 → Toggle device toolbar

## 🎬 What Happens When You Test

### 1. Open Page (0-2 seconds)
- Page loads
- Title fades in from top
- Progress indicator slides in
- Sections fade in one by one

### 2. Fill Fields (ongoing)
- Type in field → validation happens
- Valid → green checkmark appears
- Invalid → red X appears
- Progress bar extends

### 3. Complete Level (per level)
- All fields filled
- Step turns green with checkmark
- Next step becomes active (blue)
- Progress line extends

### 4. Upload Files (per file)
- Click "Choose File"
- Select file
- Preview appears with icon, name, size
- Can remove with X button

### 5. Submit Form (final)
- Click "Submit Registration"
- Button shows "Submitting..." with spinner
- Button becomes disabled
- Form submits
- Success page appears
