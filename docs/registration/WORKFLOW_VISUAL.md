## 📊 Data Flow Summary

```
ADMIN INPUT                    SYSTEM PROCESSING              STUDENT OUTPUT
───────────                    ─────────────────              ──────────────

Course Name ──────────────────> Database Insert ──────────> Course Info
Course Code ──────────────────> Link Generation ──────────> Registration URL
Student ID Code ──────────────> QR Generation ────────────> QR Code Image
Training Center ───────────────> Course Locking ───────────> Locked Fields

STUDENT INPUT                  SYSTEM PROCESSING              STUDENT OUTPUT
─────────────                  ─────────────────              ──────────────

Personal Info ─────────────────> Validation ────────────────> Success/Error
Contact Details ───────────────> File Upload ───────────────> Confirmation
Education Data ────────────────> ID Generation ─────────────> Student ID
Documents ─────────────────────> Password Generation ───────> Password
                               > Email Sending ─────────────> Email
                               > Database Insert ───────────> Portal Access
```
